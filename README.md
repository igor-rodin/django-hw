# Фреймворк Django (семинары)

Установка и запуск

- Создать и инициализировать виртуальное окружение

    ```bash
    python -m venv .venv
    source .venv/bin/activate
    ```

- Установить нужные библиотеки

    ```bash
    pip install -r requirements.txt
    ```

- Запустить сервер

    ```bash
    cd homeproject
    python manage.py runserver
    ```
